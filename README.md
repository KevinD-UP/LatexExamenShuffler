# LaTeXExamenShuffler

## Pré-requis

- java 20
- maven 3.8.1

## Compiler le projet

Lancer depuis la racine du projet la commande `mvn package`

## Lancer le projet

Lancer depuis le répertoire du projet la commande `./latexExamenShuffler.sh`
